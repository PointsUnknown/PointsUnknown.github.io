Tutorial 00 - Installation Instructions

This tutorial will describe the necessary steps to install the software used in the tutorials that follow, as well as throughout the various Points Unknown courses.

Overview

This tutorial will walk you through the installation of the following pieces of software:

  • Google Earth Pro - Google Earth is a geobrowser that accesses satellite and aerial imagery, ocean bathymetry, and other geographic data over the internet to represent the Earth as a three-dimensional globe.

  • QGIS - A cross-platform free and open-source desktop geographic information system (GIS) application that provides data viewing, editing, and analysis.

  • Adobe Illustrator - Adobe Illustrator is a paid application, although a free 1-week trial is available through Adobe. Typically, we focus our teaching around open-source tools. Unfortunately, open-source competitors fall short in a lot of the exercises we will take you through on Saturday. Installing Google Earth Pro on a Mac

  1. Go to https://www.google.com/earth/download/gep/agree.html

  2. Click Agree and Download circled in red to begin the download


  3. Locate and open the GoogleEarthProMac-Intel.dmg file, double-click to open the installer.


  4. Follow the steps to install the software.


Installing QGIS on a Mac

  1. Go to http://www.qgis.org/en/site/forusers/download.html and download QGIS for your platform.

  2. On the next window click the link circled in red to begin the download.

  3. Locate the QGIS-2.16.dmg file, double-click to open. You will see the following files inside.

  4. Before install, open and read 0 READ ME BEFORE INSTALLING.rtf to make sure your machine has the right setup

  5. Double-click or right click on 1 GDAL Complete.pkg to open and install the package.

  6. Click OK when prompted.

  7. Follow the steps to install the software.

  8. Repeat Steps 5 through 7 to install 2 NumPy.pkg, 3 matplotlib.pkg, and 4 Install QGIS.pkg.

Installing Adobe Illustrator on a Mac

  1. Go to http://www.adobe.com/products/illustrator.html and click on Free Trial

  2. Locate and open the Illustrator_Installer.dmg file.

  3. Double-click the app icon to install the application

  4. Sign in with your Adobe ID. Click Sign Up to create an Adobe ID, if you do not have an existing one. (Please sign-up for the 1-week trial. If you have already completed a 1-week trial with the Adobe Suite, please use a different email address to start a trial, or come see Michael Krisch 12-3pm Tuesday for assistance.)

Installing Google Earth Pro on Windows 10

  1. Go to https://www.google.com/earth/download/gep/agree.html

  2. Click Agree and Download circled in red to begin the download

  3. Unless you have certain AdBlock software installed, it will automatically detect you have a Windows 10 computer and give you the correct installer. Download it and run the installer



  4. Agree to let software be installed

  5. Launch the Google Earth Pro application from your desktop icon or start menu to begin using the application

Installing QGIS on Windows 10

  2. Download QGIS Standalone Installer Version 2.16 (64 bit)
